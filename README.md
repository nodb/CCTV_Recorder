# CCTV_Recorder 📹

## ✍ 소개
- 실시간 CCTV 영상을 녹화하는 프로그램입니다.
- [천안시 교통 CCTV의 RTSP 주소](https://www.data.go.kr/data/15063717/fileData.do)를 통해 확보한 실시간 CCTV 영상을 예시로 합니다.

## 🗂️ 사용 기술
- Python
- OpenCV

## 💻 기능
- 키보드 기능
  - Space : 녹화 시작/중지
  - q : 설정 리셋
  - w : 명도 +
  - e : 명도 +
  - r : 대비 -
  - t : 대비 +
  - ESC : 프로그램 종료
- 마우스 기능
  - X 버튼 : 프로그램 종료
